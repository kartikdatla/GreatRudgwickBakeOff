## Installation

```bash

cd backend
npm install

# 2. Initialize the database
npm run init-db

# 3. Create an admin user
npm run create-admin
# Follow the prompts to create your admin account

# 4. Install frontend dependencies
cd ../frontend
npm install
```

## First Steps

1. **Open browser** to http://localhost:3000
2. **Login** with your admin credentials
3. **Draw a theme** - Click "Draw Theme" to select this month's theme
4. **Create accounts** - Register additional users as Bakers, Judges, or Spectators
5. **Start competing!**

## User Roles

- **Admin** - Draw themes, lock weeks, reveal scores
- **Baker** - Submit photo entries
- **Judge** - Score submissions anonymously
- **Spectator** - View and enjoy!
